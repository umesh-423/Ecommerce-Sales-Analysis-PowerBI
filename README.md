# 📊 E-Commerce Sales Performance Dashboard

## 📌 Project Overview

This project analyzes e-commerce sales data to uncover key business insights related to revenue, profitability, product performance, and customer segments. The goal is to build an interactive dashboard that helps stakeholders make data-driven decisions.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top-performing products and categories
* Understand regional sales distribution
* Detect seasonal trends in sales
* Evaluate profitability across categories

---

## 🛠️ Tools & Technologies

* **Python (Pandas, Matplotlib)** – Data cleaning and analysis
* **Power BI** – Dashboard creation and visualization
* **CSV Dataset** – Superstore sales data

---

## 📂 Dataset

The dataset contains ~9,900+ transactions with the following key fields:

* Order Date
* Sales
* Profit
* Category & Sub-Category
* Product Name
* Region
* Segment

---

## 🔍 Data Analysis Process

### 1. Data Cleaning (Python)

* Removed duplicates
* Handled data types (converted dates)
* Created new features (Year, Month, Month-Year)

### 2. Exploratory Data Analysis

* Monthly sales trend
* Sales by category and region
* Top 10 products by revenue
* Customer segment analysis
* Profitability analysis

### 3. Dashboard Development (Power BI)

Created an interactive dashboard including:

* KPI Cards (Total Sales, Total Profit, Orders, Avg Order Value)
* Monthly Sales Trend (Line Chart)
* Sales by Category (Bar Chart)
* Sales by Region (Donut Chart)
* Top 10 Products by Sales
* Profit by Category
* Interactive Filters (Region, Segment, Category, Date)

---

## 📈 Key Insights

* 📌 **Technology category generates the highest profit**, making it the most valuable segment
* 📌 **Furniture shows low profitability**, indicating potential pricing or discount issues
* 📌 **Sales peak during year-end (Nov–Dec)**, showing strong seasonal trends
* 📌 **Top products contribute a significant portion of total revenue**
* 📌 **Consumer segment drives the majority of sales**
